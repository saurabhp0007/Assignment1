## Getting Started

Follow these steps to get the project up and running on your local machine:

```bash
# 1. Clone the Repository
git clone git@github.com:saurabhp0007/Assignment1.git
cd Assignment1

# 2. Install Dependencies

## Backend
cd backend
npm install

## Frontend
cd frontend
npm install

# 3. Set Up the Backend
# Ensure you have the backend server running. The backend is built with Express.js and serves as the API for managing help center cards.

## Running the Backend Server
cd backend
npm run dev

# The backend server will run on http://localhost:4000.

# 4. Set Up the Frontend
# Ensure you have the frontend server running. The frontend is built with React and interacts with the backend API.

## Running the Frontend Server
cd frontend
npm run dev

# The frontend will be accessible at http://localhost:5173.

# 5. Running the Application
# Once both the backend and frontend servers are running, open your browser and navigate to:

http://localhost:5173

# You should see the Help Center application running.
